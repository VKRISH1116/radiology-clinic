Radiology Clinic Website — Onboarding to Deployment

A learning-first journey through the full software development lifecycle. We build a real diagnostic-clinic web app and learn every stage of how software reaches the market — one phase at a time.

Stack: Java · Spring Boot · React · PostgreSQL Deploy target: Localhost now → AWS later Method: Understand the "why" before the "how" Owner: You (mentored daily)

What we're building

A website for a radiology / diagnostic clinic in India that lets patients book appointments, register details with consent, and securely download reports — replacing today's manual, phone-based process. Staff manage bookings and upload reports; admins oversee activity and track referring-doctor payouts (a percentage of each visit's bill that varies by doctor, scan type, or amount range).

🧑‍⚕️ Patient

Register, book scans, give consent, view status, download reports securely.

🖥️ Front-desk staff

Daily appointment list, register walk-ins, upload reports, update status.

📊 Admin

Dashboard, manage referring doctors, track referral amounts & payouts.
