# 🩻 Radiology Clinic Website

A learning-first, full-stack project: a web app for a radiology / diagnostic clinic in India where
patients book scans, register with consent, and download reports securely; staff manage bookings and
upload reports; and admins track referring-doctor payouts.

Built **while learning the full software development lifecycle** — one phase at a time.

## Tech stack
| Layer | Technology | Why |
|-------|-----------|-----|
| Backend | Java 21 · Spring Boot | Industry-standard, batteries-included framework for REST APIs |
| Frontend | React | Component model + rich ecosystem for interactive UIs |
| Database | PostgreSQL | Powerful, free, open-source relational DB |
| Deploy | Localhost → AWS (long term) | Learn locally first, then cloud |
